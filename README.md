### Keybinds:
My Super Key has been changed to ALT

| Action        |  Key Binds    |
| ------------- | ------------- |
| ATL Q         |     Terminal  |
| ALT X         |   Kill Active |
| ALT 0         |  Exit Hyprland|
| ALT W         |   File Manager|
| ALT F         |Toggle Floating|
| ALT M         |      Menu     |
| ALT J         |  Toggle Split |
